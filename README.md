# 📚 Smart Edu Planner

Smart Edu Planner is a web-based study management application built using **HTML, CSS, and JavaScript**.  
It helps students organize subjects, plan schedules, manage tasks, and track study progress efficiently through an intuitive dashboard.

## ✨ Features

### 📊 Dashboard Overview
View total subjects, pending tasks, study time, and progress in one place.

### 📚 Subject Management
Add subjects with priority levels and weekly study hours.

### 🗓 Schedule Planner
Plan study schedules for different days and time slots.

### ✅ Task Manager
Manage assignments, exams, and revision tasks with deadlines.

### 📈 Progress Analytics
Visual charts showing subject distribution and task trends.

### 🌙 Dark / Light Mode
Toggle between themes for comfortable viewing.

### 💾 Local Storage Support
All planner data is stored in the browser using LocalStorage.

### 📤 Export Data
Download planner data as a backup file.

### 🗑 Reset Data
Clear all saved planner data instantly.
